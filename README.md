# Apple-beep

Fresh start in progress.

## Phase 1: Pages foundation
- Added a robust GitHub Pages deploy workflow with preflight checks.
- Added a weekly Pages health-check workflow.
- Added a minimal `site/` artifact with a production entry page and custom 404 page.

## One-time setup in GitHub
1. Open **Settings -> Pages** in this repository.
2. Under **Build and deployment**, set **Source** to **GitHub Actions**.
3. Run the `Deploy Profile Site` workflow once.
